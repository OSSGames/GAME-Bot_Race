About Bot Race

In this game you control a tiny robot through a big factory maze. You can program the robots movement with 5 cards and help him to reach its next goal.
Be careful though the factory is full of dangerous traps.

The Interface

The game interface concists of 4 parts:

Gameboard:

This is the board where the robots will be shown and moved on. You can zoom in and out via a menu entry or by using the mouse wheel.

Carddeck:

The carddeck shows all available cards for the current round you can use to program the robot. The player receives 9 Cards, minus 1 card for each damage token it receives.
Cards can be placed in the program slots bei either double-click or drag and drop.

Statuswindow:

In this window additional information about the game are shown.

Robot Control Panel:


The control panel is the main element in the game.
It allows the player to program the the robots via 5 Cards which can be taken from the robots card deck (by double-click or drag and drop).
In addition the next flag goal is shown in the upper left and the remaining lifes in the bottom left.

The current damage token are shown on the right side. If the player receives more than 4 damage tokens for each new token one of the cards will be locked (shown in grey).
These locked cards can't be changed and won't be returned to teh deck after the phase is over.

The button in the upper right are used to tell the game that you have finished programing your robot. The button on the bottom right announce the power down.
The power down can only be used before the programming is finished and will power down the robot after the current round is played completely.

The Cards

There exist 6 different cards that can be used to program the robot. Each card has in addition a priority printend on it.
In the case that 2 or more robots whould push each other the card with the highest priority will be executed first.

 Rotates the robot 90° to the left

Rotates the robot 90° to the right

Rotates the robot 180°

Moves the robot 1 tile

Moves the robot 2 tiles

The Game Phases

Each round is played in several phases one after each other.
  • Programming phase
  • Execute card [1-5] (only one at a time)
    • Move Robot according to the current card
    • Move Conveyor belts Express/Normal/Water drain
    • Move Express Belts only
    • Move Pusher
    • Move Crusher
    • Shoot Lasers
  • Check if some robots won
  • Resurrect dead robots

The Board Elements

Each board is created from a set of board elements. Each element has a different function.
Sometimes the elements are not active in each phase, small red numbers indicate the phase in which the elements will be executed.

Floors

Normal Floor. Nothing happens on this tile.
Automatic Pit. Activated on certain phases. The robot falls down whenever the pit is activated and thus open.
Conveyor Belt Left. Transports the robot. The green one only 1 Tile per phase the Blue Express Belts 2 Tiles per phase.
The curved Belts also rotate the robot when he gets moved onto it by another belt.
Edge of the Board. The robot falls down and is dead.
Radioactive hazard. Every time the robot moves over it. It receives 1 damage token.
Hazard Pit. The robot falls down and will be killed.
Oil. Whene the robot moves on it, it slides until the next tile without oil is reached or another robot/ a wall blocks it. If the Robot starts its move in this tile the wheels will spin. Thus A move 2 or higher is necessary to get out.
Normal Pit. The robot falls down and will be killed.
Randomizer. When the robots moves onto this tile the next register card will be replaced with a random one.
Repair Tile. When the robot ends its last move (Phase 5) on this tile it will repair 1 damage token. Also whenever it moves onto this tile the archive marker is placed here.
Repair and Options Tile. When the robot ends its last move (Phase 5) on this tile it will repair 2 damage token or receive an option card. Also whenever it moves onto this tile the archive marker is placed here.
Teleporter. Whenever the robot moves onto this tiles, it will be teleported 2 tiles further away.
Water. Hinders the robot movement.  If the Robot starts its move in this tile the wheels will spin. Thus A move 2 or higher is necessary to get out.
Water drain. Transports the robot 1 Tile into the direction of the arrow in the conveyor belts phase. Also all rules of the normal water tiles apply here.
Water pit. The robot falls down and will be killed.
Gears left. The robot will be rotated counterclockwise at the end of the register phase.
Gears right. The robot will be rotated clockwise at the end of the register phase.

Walls

Normal wall. Hinders the robot to move through.
Laser wall. Shoots 1,2 or 3 lasers. Whenever a robot ends its turn in a laser it receives 1,2 or 3 damage tokens.
Pusher . At the end of each turn the pusher (if activated in this phase) will push the robot 1 tile away. If the robot can't be pushed, it receives 1 damage token.
Crusher. At the end of each turn the crusher (if activated in this phase) kills the robot.
Edge. The robot falls down if it moves in the direction of the arrow and receives 2 damage tokens. The Edge acts as a wall in the opposite direction.
Ramp. The ramp is used to climb up an endge again. A move 2 or higher card is necessary to get up the ramp
Fire wall. If the robot moves through the fire (if activated) it receives 1 additional damage token.

The Game Modes

Different game modes are available for this game. Each mode has different goals.

Hunt the Flag

In this mode each robot has to get to the next flag on the board. As soon as one robot reached the last flag the game is over.

Death or Alive

In this mode no flags are available. Instead each robot must try to kill as many other robots as possible the first Robot that reached 10 kills wins the game.

King of The Flag

The goal is to catch the flag and hold it as long as possible. Once you have the flag you can move around with it. If you die the flag will be dropped your robot died or placed back to the starting point if you jumped into a pit. Each Card phase you gain 0.2 points, once you have reached to points to win (default is 10) you have won the game.

King of the Hill

Simmilar to King of the Flag, but instead of running around with a flag you need to stay as long as possible at the flag point. Each Card phase you gain 0.2 points, once you have reached to points to win (default is 10) you have won the game.

Tips and Tricks

When you prohram your robot think about what other robots might do. Robots can push each other and you might not end up where you wanted to be.
Remember higher priority cards are always executed first.

If you are unsure where you end up after all factory elements are executed you can enable a card simulator (Help->Show Simulator) this will calculate your robots movement if no other robot interferes.
